# Random Password Generator

**Автор:** [Ваше Имя Фамилия]
**Язык:** Python 3.x

## Описание

Простое кроссплатформенное приложение для генерации случайных паролей с графическим интерфейсом. Позволяет настраивать длину и состав символов (цифры, буквы разного регистра, спецсимволы). История сгенерированных паролей сохраняется в файл `history.json` для последующего использования.

## Технологии

*   **Python:** Язык программирования.
*   **Tkinter:** Библиотека для создания GUI.
*   **random:** Стандартная библиотека для генерации случайных чисел.
*   **json:** Для сохранения истории.
*   **Git:** Система контроля версий.

## Установка и запуск

1.  Убедитесь, что установлен Python 3.
2.  Скачайте или склонируйте репозиторий:
    ```bash
    git clone https://github.com/ваш_юзер/название_репо.git
    ```
3.  Перейдите в папку проекта:
    ```bash
    cd название_репо
    ```
4.  Запустите скрипт:
    ```bash
    python password_generator.py
    ```
*(или `python3`, в зависимости от вашей системы).*

## Использование приложения

1.  **Длина:** Установите желаемую длину пароля с помощью ползунка (от 4 до 32 символов).
2.  **Типы символов:** Отметьте галочками нужные типы символов (A-Z, a-z, 0-9, спецсимволы).
3.  **Генерация:** Нажмите кнопку «Сгенерировать». Пароль появится в поле ввода.
4.  **Копирование:** Нажмите «Копировать», чтобы поместить пароль в буфер обмена.
5.  **История:** Все сгенерированные пароли отображаются в таблице ниже. Они сохраняются между запусками программы.
